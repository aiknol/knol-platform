System Architecture

Microservice architecture with async event-driven processing, multi-tenant isolation, and adaptive retrieval.

Service Details

Gateway Service AxumOSS

Central API entry point. Handles JWT/API-key authentication, Redis-backed sliding window rate limiting, tenant context injection via PostgreSQL RLS, scope resolution, feature gating across 4 tiers (OSS/Starter/Pro/Enterprise), PII detection/redaction, and Prometheus metrics collection.

Write Service OSS

Ingestion service with fast-ACK pattern. Stores raw episodes to PostgreSQL, then emits a MemoryWriteEvent to NATS JetStream for async background processing. Supports single and batch writes with idempotency.

Retrieve Service OSS

Adaptive retrieval router that classifies query intent (preference, temporal, relational, general) and selects the optimal strategy. Combines vector similarity (pgvector HNSW), BM25 full-text, graph traversal, and scope cascade, fusing results via Reciprocal Rank Fusion.

Graph Service OSS

NATS JetStream consumer. On receiving extraction events, calls Claude Haiku for entity/relationship extraction from text. Upserts bi-temporal entities and edges with deduplication and merge logic.

Admin Service Enterprise

Full CRUD for memories with audit diff tracking. Governance rules, retention policy management, audit log browser, and memory simulation/replay for testing retrieval strategies.

Jobs Service Enterprise

Tokio cron-scheduled background jobs: memory importance decay, duplicate detection, episodic→semantic consolidation, conflict detection (word-overlap similarity), and automated conflict resolution.

Billing Service Enterprise

Usage event aggregation, plan limits enforcement, and tier checking. Tracks memory storage, search queries, and API calls per tenant against their subscription plan.

Ingest Service Enterprise

Connector framework with trait-based plugins. Ingests data from external sources (Slack messages, GitHub issues, email) via webhooks and polling, normalizes content, and feeds into the write pipeline.
